# Reporter

This is a RubyGem for reading and modifying [Reporter](http://www.reporter-app.com/) reports and snapshots.

## Installation

Add this line to your application's Gemfile:

    gem 'reporter'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install reporter
